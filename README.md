# Carrefour QA Automation Framework

Automation framework developed for a Senior QA Automation technical challenge.

## Technologies

Mobile Automation
- WebdriverIO
- Appium
- Mocha
- Chai
- Page Object Model
- Allure Reports

API Automation
- Node.js
- Supertest
- Mocha
- Chai

## Project Structure

carrefour-qa-automation
│
├── api-tests
├── mobile-tests
├── package.json
├── .gitignore
└── README.md

## Running the tests

Install dependencies:

npm run install:all

Run API tests:

npm test

Run mobile tests:

npm run test:mobile
